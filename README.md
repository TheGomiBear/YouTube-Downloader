# YouTube-Downloader
A YouTube downloader tool I made for fun :P

# INSTRUCTIONS TO SET UP THE DOWNLOADER

1. Download Node.JS using links below:
x64: nodejs.org/dist/v25.9.0/node-v25.9.0-x64.msi
ARM64: nodejs.org/dist/v25.9.0/node-v25.9.0-arm64.msi
x86: nodejs.org/dist/v22.22.2/node-v22.22.2-x86.msi

2. Download yt-dlp.exe using the link below:
github.com/yt-dlp/yt-dlp/releases/latest/download/yt-dlp.exe

3. Place yt-dlp.exe file in the folder this tool is located.

4. If YouTube isn't supported in your country, then download GoodbyeDPI using the link below:
github.com/ValdikSS/GoodbyeDPI/releases/download/0.2.3rc3/goodbyedpi-0.2.3rc3-2.zip
  4.1. Extract the .zip file
  4.2. Launch the 2_any_country.cmd file.
  4.3. 2_any_country_dnsredir.cmd is the alternative way.

4. And also, do NOT delete yt-dlp.conf file! It is a configuration
file for yt-dlp. Through it, yt-dlp uses Node.JS, because without it,
the tool won't work.

# INSTRUCTIONS TO USE THIS YOUTUBE DOWNLOADER

1. Paste the link into the command prompt you opened using the
tool file.

2. Choose the format you need.

3. Wait till whatever you need to download downloads.
